# Query Generation for Phase 1

## Query Strategy

Generate 3-5 search queries optimized for finding intelligence sources.

### Query Patterns by Goal

**For Product Research:**
- `"best [product] [year] reddit"` - Community recommendations
- `"[product] buying guide"` - Comprehensive guides
- `"[product] vs [product]"` - Comparison insights
- `"[product] specs to look for"` - Technical guidance

**For How-To/Care:**
- `"how to [action] guide"` - Step-by-step guides
- `"[topic] tips reddit"` - Community tips
- `"[topic] mistakes to avoid"` - Cautionary advice
- `"[topic] for beginners"` - Entry-level guidance

**For General Information:**
- `"[topic] explained"` - Educational content
- `"[topic] facts"` - Factual information
- `"what to know about [topic]"` - Overview content

### Query Optimization

1. **Add "reddit" or forum names** - Gets community discussions
2. **Add current year** - Gets recent information
3. **Use specific terminology** - More relevant results
4. **Avoid brand names initially** - Get unbiased recommendations first
