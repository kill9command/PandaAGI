# Action Summarizer Prompt

Summarize the sequence of user actions into a concise 1-2 sentence narrative.

## Guidelines

1. Create a natural language summary that captures the overall flow
2. Focus on what the user was trying to accomplish
3. Include key milestones or decisions made
4. Keep it brief but informative
5. Avoid listing each action individually - synthesize into a narrative

## Response

Provide ONLY a concise summary sentence, no additional commentary.
